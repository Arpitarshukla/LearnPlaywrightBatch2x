# VS Code Default Keyboard Shortcuts — macOS

## General

| Shortcut | Action |
|---|---|
| `Cmd+Shift+P` / `F1` | Show Command Palette |
| `Cmd+P` | Quick Open, Go to File |
| `Cmd+Shift+N` | New window/instance |
| `Cmd+W` | Close window/instance |
| `Cmd+,` | User Settings |
| `Cmd+K Cmd+S` | Keyboard Shortcuts |

## Basic Editing

| Shortcut | Action |
|---|---|
| `Cmd+X` | Cut line (empty selection) |
| `Cmd+C` | Copy line (empty selection) |
| `Opt+↑ / ↓` | Move line up/down |
| `Shift+Opt+↓ / ↑` | Copy line up/down |
| `Cmd+Shift+K` | Delete line |
| `Cmd+Enter` | Insert line below |
| `Cmd+Shift+Enter` | Insert line above |
| `Cmd+Shift+\` | Jump to matching bracket |
| `Cmd+]` / `Cmd+[` | Indent/outdent line |
| `Home` / `End` | Go to beginning/end of line |
| `Cmd+↑` / `Cmd+↓` | Go to beginning/end of file |
| `Ctrl+PgUp` / `Ctrl+PgDn` | Scroll line up/down |
| `Cmd+PgUp` / `Cmd+PgDn` | Scroll page up/down |
| `Opt+Cmd+[` | Fold (collapse) region |
| `Opt+Cmd+]` | Unfold (uncollapse) region |
| `Cmd+K Cmd+[` | Fold all subregions |
| `Cmd+K Cmd+]` | Unfold all subregions |
| `Cmd+K Cmd+0` | Fold all regions |
| `Cmd+K Cmd+J` | Unfold all regions |
| `Cmd+K Cmd+C` | Add line comment |
| `Cmd+K Cmd+U` | Remove line comment |
| `Cmd+/` | Toggle line comment |
| `Shift+Opt+A` | Toggle block comment |
| `Opt+Z` | Toggle word wrap |

## Multi-cursor & Selection

| Shortcut | Action |
|---|---|
| `Opt+Click` | Insert cursor |
| `Opt+Cmd+↑ / ↓` | Insert cursor above/below |
| `Cmd+U` | Undo last cursor operation |
| `Shift+Opt+I` | Insert cursor at end of each line selected |
| `Cmd+L` | Select current line |
| `Cmd+Shift+L` | Select all occurrences of current selection |
| `Cmd+F2` | Select all occurrences of current word |
| `Ctrl+Shift+Cmd+→ / ←` | Expand / shrink selection |
| `Shift+Opt+Cmd+↑ / ↓` | Column select up/down |
| `Shift+Opt+Cmd+← / →` | Column select left/right |

## Search & Replace

| Shortcut | Action |
|---|---|
| `Cmd+F` | Find |
| `Opt+Cmd+F` | Find & Replace |
| `Cmd+Shift+F` | Find in files |
| `Cmd+Shift+H` | Replace in files |
| `Cmd+G` / `Cmd+Shift+G` | Find next/previous |
| `Opt+Enter` | Select all occurrences of Find match |
| `Cmd+D` | Add selection to next Find match |
| `Cmd+K Cmd+D` | Move last selection to next Find match |

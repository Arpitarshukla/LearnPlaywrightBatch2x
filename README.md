# Learn Playwright Batch 2x

A hands-on repository for learning Playwright, starting with JavaScript fundamentals and progressing to Playwright automation.

## Chapters

- **Chapter 01: Basics** — JavaScript basics, setup verification, and Playwright intro (`01_Basics.js` — `04_HotCode.js`)
- **Chapter 02: JavaScript Concepts** — Core JavaScript concepts (`05_JS_Basics.js`)
- **Chapter 03: Identifiers & Literals** — Identifier rules, comments, and VS Code shortcuts (`06_Identifier_rules.js`, `07_Identifier_Part2.js`, `08_Comments.js`)

## Getting Started

1. Clone the repo
2. Run any `.js` file using Node.js:
   ```bash
   node <filename>
   ```
3. Refer to VS Code shortcut guides in `Chapter_03/` for editor tips.

## Prerequisites

- [Node.js](https://nodejs.org/)
- [VS Code](https://code.visualstudio.com/)

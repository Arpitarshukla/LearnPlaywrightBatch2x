# Learn Playwright Batch 2x

A hands-on repository for learning Playwright automation testing, starting with JavaScript fundamentals.

## Chapters

- **Chapter 01 - Basics**: JavaScript basics, setup verification, and Playwright intro
  - Files: `01_Basics.js`, `02_JS.js`, `03_JS_Verify_Setup.js`, `04_HotCode.js`

- **Chapter 02 - JavaScript Concepts**: Core JavaScript concepts
  - Files: `05_JS_Basics.js`

- **Chapter 03 - Identifier & Literals**: JavaScript identifier rules, literals, VS Code shortcuts, and comments
  - Files: `06_Identifier_rules.js`, `07_Identifier_Part2.js`, `08_Comments.js`, identifier rules cheat-sheet, VS Code keyboard shortcuts (mac & windows)

- **Chapter 04 - Javascript Concepts**: Variables (`var`, `let`, `const`), functions, hoisting, and scope
  - Files: `09_var_let_const.js`, `10_functions.js`, `11_var_explained.js`, `12_let_people_love.js`, `13_const_explained.js`, `14_var_functionscope.js`, `15_let_scope.js`, `16_Hoisting.js`, `17_hoisting_fn.js`, `18_let_hoisting.js`, `19_let_hoisting_block.js`, `20_let_const.js`, `21_Jr_QA.js`

- **Chapter 05 - Literal**: JavaScript literals (null, undefined, numbers, strings, template literals, backticks)
  - Files: `22_Literal.js`, `23_null_undefined.js`, `24_null.js`, `25_Literal_All.js`, `26_Literal_Number_all.js`, `27_String.js`, `28_Template_Literal.js`, `29_Backtick_single_double.js`

- **Chapter 06 - Operators**: JavaScript operators (assignment, arithmetic, comparison, logical, ternary, nullish, increment/decrement, type, string concatenation)
  - Files: `30_Assignment_operators.js`, `31_Arithmatic_operators.js`, `32_Modulus_OP.js`, `33_Exponential_OP.js`, `34_IQ.js`, `35_Comparison_Op.js`, `36_Comparison_Strict_loose.js`, `37_IQ_Loose_Strict.js`, `38_Confusing_Comparison.js`, `39_Logical_Op.js`, `40_String_Con_Op.js`, `41_Ternary_op.js`, `42_Type_Op.js`, `43_Incre_Decre_Op.js`, `44_Null_Op.js`, `45_Post_increment.js`, `46_IQ_INCREMENT_D.js`, `47_ADVANCE_ID.js`

## Prerequisites

- [Node.js](https://nodejs.org/)
- [VS Code](https://code.visualstudio.com/)
- Playwright

## Getting Started

1. Clone the repo:
   ```bash
   git clone <repo-url>
   ```
2. Run any `.js` file using Node.js:
   ```bash
   node <filename>
   ```
3. Refer to VS Code shortcut guides in `Chapter_03/` for editor tips.
